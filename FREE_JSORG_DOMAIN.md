# 🆓 Get FREE nexusrevive.js.org Domain

## Why js.org?
- ✅ 100% FREE forever
- ✅ More professional than .vercel.app
- ✅ Perfect for JavaScript/tech projects
- ✅ Trusted by 1000+ open source projects
- ✅ Custom DNS, SSL included

---

## Step-by-Step Guide:

### Step 1: Fork js.org Repository
1. Go to: https://github.com/js-org/js.org
2. Click "Fork" (top right)
3. Wait for fork to complete

### Step 2: Edit cnames_active.js
1. In YOUR fork, open file: `cnames_active.js`
2. Click pencil icon (Edit)
3. Find alphabetical spot for "nexusrevive"
4. Add this line:

```javascript
  "nexusrevive": "nexus-revive.vercel.app",
```

5. Commit changes

### Step 3: Submit Pull Request
1. Click "Pull requests" tab
2. Click "New pull request"
3. Click "Create pull request"
4. Title: "Add nexusrevive.js.org"
5. Description: "Nexus Revive - File recovery and conversion SaaS"
6. Submit

### Step 4: Wait for Approval
- Usually: 1-3 days
- Maintainers review
- Once approved, your domain is live!

### Step 5: Configure in Vercel
1. Vercel Dashboard → Your Project → Settings → Domains
2. Add: `nexusrevive.js.org`
3. Vercel will auto-configure

### Step 6: Update Environment Variables
```bash
NEXTAUTH_URL=https://nexusrevive.js.org
NEXT_PUBLIC_APP_URL=https://nexusrevive.js.org
```
